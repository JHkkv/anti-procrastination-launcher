> 「拖延症启动器」设计参考库 - 动画、过渡与治愈系反馈

---

## 1. 微交互设计原则

### 1.1 核心理念

微交互是用户操作与系统反馈之间的微观对话。对于拖延症App，微交互需要：

- **温柔反馈** — 不过度庆祝，也不冷淡
- **自然节奏** — 模拟自然界的速度（花开、云动、水流）
- **减少焦虑** — 避免快速闪烁、突然弹出等刺激性动画
- **暗示成长** — 每个动画都应暗示"你在前进"

### 1.2 动画时长参考

```css
:root {
  /* 动画时长层级 */
  --duration-instant: 100ms;    /* 微反馈：颜色变化、透明度 */
  --duration-fast: 150ms;       /* 快速响应：按钮态、hover */
  --duration-normal: 300ms;     /* 标准过渡：弹窗、面板 */
  --duration-slow: 500ms;       /* 柔和过渡：页面切换 */
  --duration-breath: 800ms;     /* 呼吸感：完成动画、奖励 */
  --duration-scene: 1200ms;     /* 场景切换：全屏过渡 */
  --duration-grow: 2000ms;      /* 生长动画：植物/花园成长 */

  /* 缓动函数 - 治愈系 */
  --ease-out-soft: cubic-bezier(0.25, 0.46, 0.45, 0.94);
  --ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
  --ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);
  --ease-breath: cubic-bezier(0.4, 0, 0.2, 1);
  --ease-gentle-bounce: cubic-bezier(0.34, 1.3, 0.64, 1);
}
```

### 1.3 缓动函数选择指南

```
[缓动函数使用场景]

函数                曲线特征            使用场景
------------------------------------------------------------------
ease-out-soft       柔和减速            hover效果、颜色过渡
ease-out-expo       快速起步，极慢收尾  滑入/滑出、展开
ease-spring         弹性回弹            勾选反馈、收藏动画
ease-breath         呼吸般均匀          呼吸灯、脉动效果
ease-gentle-bounce  温和弹跳            任务完成、获得奖励

避免使用：
- linear（机械感）
- ease-in（操作后才加速，感觉迟钝）
- steps（卡顿感）
```

```javascript
// Lottie Web 集成示例
import lottie from 'lottie-web';

// 任务完成动画
function playCompletionAnimation(container, animationData) {
  const anim = lottie.loadAnimation({
    container,
    renderer: 'svg',
    loop: false,
    autoplay: true,
    animationData,
  });

  anim.setSpeed(0.8);  // 稍慢于默认，更治愈
  anim.addEventListener('complete', () => {
    anim.destroy();
  });

  return anim;
}

// 呼吸循环动画(用于等待/专注)
function playBreathingLoop(container, animationData) {
  return lottie.loadAnimation({
    container,
    renderer: 'svg',
    loop: true,
    autoplay: true,
    animationData,
  });
}
```

## 10. 动画性能优化

```css
/* 性能优化原则 */

/* 1. 使用will-change提前通知浏览器 */
.animated-element {
  will-change: transform, opacity;
}

/* 动画结束后移除will-change */
.animated-element--done {
  will-change: auto;
}

/* 2. 只动画化合成层属性 */
/* 推荐：transform, opacity, filter */
/* 避免：width, height, top, left, margin, padding */

/* 3. 使用contain限制重排 */
.card {
  contain: layout style;
}

/* 4. 尊重用户的减少动画偏好 */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }

  .loading-dots,
  .loading-petals,
  .loading-wave,
  .breath-circle {
    /* 用静态替代方案 */
    animation: none !important;
  }

  .loading-dots::after {
    content: '加载中...';
    color: var(--color-text-secondary);
  }
}
```
