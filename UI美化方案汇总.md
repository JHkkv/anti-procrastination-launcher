## 通用改进（所有方案都建议加入）

1. **引入 LXGW WenKai 字体** — 免费开源书法体，直接提升标题质感
2. **等宽数字字体** — DM Mono 或 JetBrains Mono，防止数字跳动抖动
3. **按钮按下弹性** — `scale(0.97)` + spring easing
4. **步骤完成闪光** — 背景色短暂闪为极淡主色再渐回
5. **prefers-reduced-motion** — 所有动画支持无障碍降级
