## 1. 色彩与情绪的基础关系

### 1.1 色彩心理学核心原理

色彩通过视觉神经直接影响下丘脑，进而影响情绪和生理反应。对于拖延症用户，色彩选择的核心目标是：

- **降低焦虑** — 避免刺激性色彩
- **提供安全感** — 使用温暖但不灼热的色调
- **激发行动力** — 适度的正向激励色
- **减少自我批判** — 柔和的语义色，避免强烈的"红=错"暗示

### 1.2 基础色彩的情绪映射

```
[暖色系]

红色(高饱和)  →  紧迫、焦虑、警告    ✗ 拖延症App禁用
红色(低饱和)  →  温暖、关怀、包容    ✓ 作为点缀
橙色(高饱和)  →  活力、刺激、急躁    ✗ 避免
橙色(低饱和)  →  温暖、舒适、亲切    ✓ 作为辅助色
黄色(高饱和)  →  警告、紧张          ✗ 避免
黄色(低饱和)  →  明亮、轻快、希望    ✓ 大面积使用安全
米色/杏色     →  柔软、安心、家的感觉 ✓ 极佳的背景色

[冷色系]

蓝色(低饱和)  →  平静、信任、辽阔    ✓ 信息色
绿色(低饱和)  →  成长、安全、自然    ✓ 主色调首选
紫色(低饱和)  →  神秘、创意、放松    ✓ 装饰色
灰色(暖灰)    →  中性、稳定、成熟    ✓ 文字色

[中性色]

白色(微暖)    →  干净、呼吸、空间    ✓ 大量使用
奶白/象牙     →  温和、不刺眼        ✓ 比纯白更治愈
```

## 2. 适合拖延症App的配色策略

### 2.1 核心原则

**温暖但不燥热** — 色温偏向暖色，但饱和度严格控制在30%以下
**安全但不无聊** — 用微妙的色彩变化制造层次，而非大面积单色
**激励但不压迫** — 完成/成就用柔和的正向色，避免竞争性色彩
**变化但不混乱** — 提供季节主题切换，但每个主题内部色彩统一

### 2.2 情绪-场景-色彩映射

```
[应用场景配色建议]

场景              情绪目标         推荐色彩方向
----------------------------------------------------------
首次打开          安全感、欢迎     暖米色背景 + 柔绿点缀
任务列表          轻松、不压迫     浅暖白底 + 低密度卡片
开始任务          鼓励、不焦虑     柔绿渐变 + 暖光效果
执行中            专注、平静       淡蓝/淡绿 + 柔和计时器
任务完成          成就感、喜悦     柔粉+金光粒子动画
休息页面          放松、恢复       紫+蓝渐变 + 自然意象
统计页面          正向、不攀比     暖绿渐变浓淡表示投入量
设置页面          中性、清晰       标准中性色系统
```

## 3. 深色模式 vs 浅色模式

### 3.1 心理影响对比

```
[浅色模式]

优势：
- 传达积极、清醒的情绪
- 符合"白天做事"的心理暗示
- 文字阅读舒适度高
- 与自然日光环境协调
- 减少孤独感和封闭感

劣势：
- 夜间使用刺眼
- 长时间注视可能疲劳

适用时段：白天、早晨、下午
适用场景：查看任务、管理日程、完成打卡

----------------------------------------------

[深色模式]

优势：
- 夜间使用不刺眼
- 创造私密、安全的空间感
- 减少蓝光刺激，有助睡眠准备
- 适合冥想、反思、回顾类功能
- 降低屏幕存在感

劣势：
- 可能强化低落情绪（对部分用户）
- 长时间使用可能增加困倦感
- 文字可读性需要更精细处理

适用时段：夜间、睡前
适用场景：晚间回顾、冥想放松、查看明日计划
```

### 3.2 深色模式配色方案

```css
/* 深色模式 - 治愈系暗色（非纯黑） */
:root[data-theme="dark"] {
  /* 背景 - 深褐灰而非纯黑 */

  --color-surface: #2A2622;          /* 卡片 - 微暖深色 */
  --color-surface-warm: #332E29;     /* 浮层 */

  /* 文字 - 柔和的浅色而非纯白 */
  --color-text-primary: #E8E0D8;     /* 主文字 - 暖白 */
  --color-text-secondary: #A09890;   /* 次要文字 */
  --color-text-tertiary: #706860;    /* 辅助文字 */

  /* 主色调 - 降低明度保持饱和 */
  --color-primary: #6B8F73;          /* 夜间抹茶 */
  --color-primary-light: #3A4A3D;
  --color-secondary: #8B6B70;        /* 夜间樱花 */
  --color-secondary-light: #4A3538;
  --color-accent: #6B8098;           /* 夜间天空 */
  --color-accent-light: #354050;

  /* 语义色 - 全部降低亮度 */
  --color-success: #6B8F6B;
  --color-warning: #8B7B60;
  --color-error: #8B6060;            /* 暗红而非亮红 */
  --color-info: #607888;

  /* 阴影 - 用发光而非投影 */
  --shadow-sm: 0 2px 8px rgba(0, 0, 0, 0.3);

  --shadow-glow-green: 0 4px 20px rgba(107, 143, 115, 0.2);

}
```

### 3.3 自动切换策略

```css
/* 跟随系统 + 时间自适应 */
@media (prefers-color-scheme: dark) {
  :root {
    /* 应用深色模式变量 */
  }
}

/* 额外：21:00-6:00 自动柔和化 */
/* 通过JS检测时间，添加 class="night-mode" */
:root.night-mode {
  --color-bg: #1E1B18;
  /* 进一步降低对比度 */
  --color-text-primary: #D8D0C8;
  /* 可叠加暖色滤镜 */
  filter: sepia(0.05) saturate(0.9);
}
```

### 4.2 季节切换实现

```javascript
// 自动检测季节
function getSeason() {
  const month = new Date().getMonth() + 1;
  if (month >= 3 && month <= 5) return 'spring';
  if (month >= 6 && month <= 8) return 'summer';
  if (month >= 9 && month <= 11) return 'autumn';
  return 'winter';
}

// 应用季节主题
function applySeasonTheme(season) {
  document.documentElement.setAttribute('data-season', season);
}

// 初始化
applySeasonTheme(getSeason());
```
